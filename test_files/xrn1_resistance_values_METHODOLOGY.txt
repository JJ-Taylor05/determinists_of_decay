METHODOLOGY NOTES: xrrna_xrn1_resistance_values.csv
=====================================================
Generated for use with pyTEISER alongside xrrna_flavivirus_3utr.fa


WHAT THESE VALUES ARE
---------------------
The xrn1_resistance column contains fractional resistance values (0.0–1.0)
representing the proportion of RNA that remains intact (i.e., resists degradation)
after in vitro Xrn1 challenge. A value of 0.95 means ~95% of the RNA withstands
Xrn1 under standard assay conditions.

This is the same scale used in published gel-quantification figures from the
Chapman lab (eLife 2014) and Dilweg et al. (RNA Biol 2020, 2021).


EVIDENCE CLASSES
----------------
Class A — Value derived directly from a published in vitro Xrn1 resistance assay
           on the same virus or a closely related strain tested explicitly.
           Applies to: DENV2, ZIKV, WNV/Kunjin.

Class B — Value estimated from:
           (a) confirmed sfRNA production in infected cells (Northern blot), AND
           (b) structural conservation with a Class A virus, AND/OR
           (c) gel band intensity descriptions in primary papers (e.g. "robust",
               "weaker than DENV2").
           Applies to: DENV1, DENV3, DENV4, YFV, JEV, SLEV, TBEV, HCV.

IMPORTANT: Class B values carry uncertainty of approximately ±0.05–0.10.
They should be treated as literature-informed estimates, not precise measurements.


WHY EXACT VALUES AREN'T UNIFORMLY AVAILABLE
--------------------------------------------
In vitro Xrn1 resistance assays produce gel images. Most papers report:
  - Presence/absence of a resistant band (qualitative)
  - "Nearly quantitative" resistance (for DENV2, ZIKV, WNV — the most-studied)
  - Percentage remaining RNA for MUTANT constructs (to show loss of function)
  - Rarely: precise WT percentages as a number in the text

Precise WT efficiency percentages are most commonly reported for:
  - DENV2 xrRNA1: ~95% (MG aptamer fluorescence assay, Chapman et al. 2014)
  - TBEV constructs: Dilweg et al. 2020 report mutant values "not exceeding 10%",
    implying WT well above that; estimated ~70–75% from gel description.

For a rigorous experiment, values should be confirmed or replaced with your own
in vitro assay data.


HOW TO USE THIS WITH pyTEISER
------------------------------
pyTEISER expects a tab-separated or CSV expression file with:
  - Column 1: sequence identifier (must match FASTA headers exactly)
  - Column 2: numeric phenotype value

For pyTEISER, create a simplified two-column version:

  sequence_id                      xrn1_resistance
  NC_001477.1_DENV1_xrRNA1         0.92
  NC_001474.2_DENV2_xrRNA1         0.95
  ...

The tool will compute mutual information between the expression values and
candidate sequence-structure seeds. A higher gradient across your 12 sequences
(range 0.61–0.95) is better for MI calculation than clustered values.

Command example:
  pyteiser_pipeline \
    --rna_fastafile xrrna_flavivirus_3utr.fa \
    --exp_values_file xrrna_xrn1_resistance_values_simple.csv \
    --seeds_file seeds_20.bin \
    --temp_folder temp/ \
    --out out/


RECOMMENDED ADDITIONAL STEP
-----------------------------
Before using these values in any publication or formal analysis, cross-reference
each entry against the original papers listed in the primary_source column.
For DENV1, DENV3, DENV4, JEV, and SLEV specifically, consider replacing Class B
estimates with values from a direct in vitro Xrn1 assay of your own.

Alternatively, the PolyA_DB / ViralZone databases may have additional
quantitative sfRNA/xrRNA characterisation data published after early 2026.
